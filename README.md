# MemeVerse

MemeVerse is like Instagram.

Overview:

Used Tech: 
- React.js
- Node.Js
- JWT Authentication
- MongoDB Atas

Feature:

- JWT Authentication
- View profile
- Upload, Delete Post
- Like, Unlike , Comment

How to use: 

go to MemeVerse directory

cd server && npm install
cd client && npm install

cd..
npm start
